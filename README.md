# nordtech-devsecops-pipeline


Projekt: M4K Gang - nordtech-devsecops-pipeline

Teamnamn: Mattej Petrovic, Carl Persson, Julia Persson, Jonny Nguyen, Oskar Palm

Beskrivning: Ett lösenordsstyrkeverktyg som analyserar hur säkert ett lösenord är genom att kontrollera längd, variation av tecken och vanliga mönster. Syftet är att hjälpa användare skapa starkare lösenord som är svårare att gissa eller knäcka.
